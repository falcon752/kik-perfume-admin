# MERN Website

A full-stack web application built with **MongoDB**, **Express**, **React**, and **Node.js**.

## Features
- User authentication
- CRUD operations
- Responsive design
- API integration

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Others: Axios, JWT

## Setup
1. Clone the repo:
   ```bash
   git clone <repo-url>
   cd <repo-folder>

Install dependencies:
npm install
cd client
npm install

Create a .env file:
ini
MONGO_URI=your_mongo_uri
JWT_SECRET=your_secret
PORT=5000

Run the app:
npm run dev
Open http://localhost:3000
